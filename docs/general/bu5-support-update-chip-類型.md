目前BU5 可以 update chip 如下
# Hub 
BU5主要產品，基本上都會透過Hub 跟其他chip 做溝通，chip 如下
- GL3523
- GL3510
- GL3590
- GL3525
## PD
BU5 新開發產品，目前第一代chip 已經開始推給客戶，第二代chip開發中，很多時候都會搭上自家Hub 推薦給客戶
也有別家PD會搭上自家Hub，如Ti , MTK等等
- GL9510
## HostBridge
主要會在GL3590裡面出現，功能為透過Hub 連結兩台電腦，可以share Mouse , KeyBoard，以及互相copy file ，有Support的系統如下
- Windows
- Mac
- Linux
- Android
## Scaler 
主要功能為控制螢幕顯示，目前有越多越多的Monitor 都會搭上Hub ，所以跟其他家Scaler 配合的案子多很多，目前合作公司如下
- MTK
- RealTek
- NovaTek
