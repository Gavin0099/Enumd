## 前緣
當初要升級GitLab 的原因是因為CodeView ，15.2.0 版本在MR(marge request) 可以 assign 給別人做code view ，之前的版本只能透過At(@) 的方式做code review ，流程沒有很順暢
## SW1 每階段工作方式
每個階段其實都是經過慢慢討論方向，往對的方向前進，雖然中間有爭執或是害怕，但是還是慢慢的在進步，並且在進步的過程有嘗到這樣做確實可以讓效率提高，也讓我們更有勇氣再去做改變
雖然現在還沒有到很完美，但是大家願意讓SW1變得更好，這才是我們前進的動力
