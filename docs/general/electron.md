## 3. 應用流程（簡述）
1. 啟動 → main.js 建立 window，注入 preload.js
1. renderer.js 在 UI 觸發時，透過 window.electron 呼叫 native API
1. native API 呼叫 addon.node，實際由 addon.cpp/addon.mm 與 DLL/Framework 溝通
1. DLL/Framework 回應查詢/更新，層層回傳至 UI
