1. Execute command line Tool
1. command prompt tool jump to m27fd directory
1. rsa_decrypt.exe “scaler encrypted bin file”
1. If the decryption is successful, the information in the red box above will be displayed.
Open the m27fd folder and there will be a temp.bin file, which is the decrypted scaler bin file

## Note
Please replace the scaler bin file with the latest version of the scaler bin file in m24fd & m27fd.
