會先詢問有接在電腦上monitor ，再去抓取對應的regedit edid info (Computer\HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Enum\DISPLAY)
詢問時間快，且可以抓到edid 基本的資訊(model name ， serial number....)
