## 條件
1. 使用command line tool 
1. 只能接一台Monitor 來測試
1. 提供一個固定檔名QCTool.ini的ini檔
## 測試方式
1. 透過command line tool 下 指令
1. Tool 會去詢問 Monitor Model & 讀取Hub Pid Vid，跟ini檔做比對來確認是否正確
1. command line tool 會顯示這次的測試結果
