Hub MP Tool
1.開啟 ini檔，輸入更新的Hub Fw Bin檔
2.開啟 HubMPTool
3.一開啟焦點就會在Serial Number Edit Box 上，直接用BarCode 掃描，確定 Serial Number 合法的話，就會自動ISP
4.燒錄成功後會在原本資料夾產生一個 Test_log.txt檔，開啟後有需要的資訊
5.重新插拔後Serail Number Edit Box 會清空，並且Hub 會initial ，就可以再次更新
