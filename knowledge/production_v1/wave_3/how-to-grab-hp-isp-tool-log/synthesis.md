## 如何抓取 HP ISP Tool 的日誌

根據提供的內容，以下是如何抓取 HP ISP Tool 日誌的步驟:

1. 在 HP ISP Tool 的資料夾中打開 GL_Hub.ini 檔案。
2. 填寫 GL_Hub.ini 檔案。
3. 開啟 HP ISP Tool。
4. 將 HP ISP Tool 資料夾中的日誌資料夾回傳給我們進行分析。


### 1. 開啟 GL_Hub.ini 檔案
在 HP ISP Tool 的安裝資料夾中找到並打開 GL_Hub.ini 檔案。這個檔案包含了 HP ISP Tool 的一些設定資訊。

### 2. 填寫 GL_Hub.ini 檔案
根據您的具體情況，填寫 GL_Hub.ini 檔案中的相關設定。這可能包括設定 API Key、指定韌體檔案路徑等。

### 3. 開啟 HP ISP Tool
啟動 HP ISP Tool 應用程式。在執行過程中，HP ISP Tool 會產生各種日誌資訊。
