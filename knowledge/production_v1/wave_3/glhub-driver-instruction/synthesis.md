# GLHub Driver Instruction


- 請安裝 Driver Ver. 2.4

### 當先前已安裝設備過濾驅動程式時
- 更新驅動程式至 Ver. 2.00

### 當先前已安裝舊版類別過濾驅動程式時

### 當同時安裝有設備過濾驅動程式和舊版類別過濾驅動程式時
- GLHub Driver v2.4 將會移除類別和設備過濾驅動程式並重新啟動。流程如下:

## GLHubDriver V2.4.0

[eToken System](code-sign/etoken-system.html)

[eToken 安全簽章系統技術說明](code-sign/etoken-安全簽章系統技術說明.html)

[ Generic USB Filter Driver ](driver/-generic-usb-filter-driver-.html)


1. 當未安裝驅動程式時，需要安裝 Driver Ver. 2.4。
2. 當先前已安裝設備過濾驅動程式時，需要移除舊版並更新至 Ver. 2.00。
4. 當同時安裝有設備過濾驅動程式和舊版類別過濾驅動程式時，GLHub Driver v2.4 會自動移除並重新啟動系統。
5. GLHubDriver V2.4.0 是當前最新版本。

這些安裝和升級步驟確保了 GLHub 驅動程式能夠順利在系統上運行。同時也介紹了相關的 eToken 簽章系統和通用 USB 過濾驅動程式的技術背景。
