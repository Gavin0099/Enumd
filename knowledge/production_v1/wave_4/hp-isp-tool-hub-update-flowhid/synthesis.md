
## HP ISP Tool Hub Update Flow(HID)

此文件描述了如何使用 HP_ISPTool.exe 軟體來更新 HP ISP Tool Hub 的韌體。

### Hub Mask Code Update 流程
1. 開啟 HP_ISPTool.exe 軟體。
1. 按下「Enter PASSWORD」按鈕，並輸入密碼「GENESYS6104」。[`HP ISP Tool Hub Update Flow(HID)`](./hp-isp-tool-hub-update-flowhid.html)
1. 輸入正確密碼後，即可進行更新。[`HP ISP Tool Hub Update Flow(HID)`](./hp-isp-tool-hub-update-flowhid.html)
1. 選擇 Hub Firmware 並選擇 Hub Rom 檔案，然後開始 ISP 更新。[`HP ISP Tool Hub Update Flow(HID)`](./hp-isp-tool-hub-update-flowhid.html)
1. 您可以看到更新成功的訊息。[`HP ISP Tool Hub Update Flow(HID)`](./hp-isp-tool-hub-update-flowhid.html)

1. 開啟 HP_ISPTool.exe 軟體。[`HP ISP Tool Hub Update Flow(HID)`](./hp-isp-tool-hub-update-flowhid.html)
1. 選擇 Hub Firmware 並選擇 Hub sig bin 檔案，然後開始 ISP 更新。[`HP ISP Tool Hub Update Flow(HID)`](./hp-isp-tool-hub-update-flowhid.html)
1. 您可以看到更新成功的訊息。[`HP ISP Tool Hub Update Flow(HID)`](./hp-isp-tool-hub-update-flowhid.html)

1. 開啟 HP_ISPTool.exe 軟體。[`HP ISP Tool Hub Update Flow(HID)`](./hp-isp-tool-hub-update-flowhid.html)
1. 選擇 Scaler Firmware 並選擇 Scaler bin 檔案，然後開始 ISP 更新。[`HP ISP Tool Hub Update Flow(HID)`](./hp-isp-tool-hub-update-flowhid.html)
1. 您可以看到更新成功的訊息。[`HP ISP Tool Hub Update Flow(HID)`](./hp-isp-tool-hub-update-flowhid.html)
