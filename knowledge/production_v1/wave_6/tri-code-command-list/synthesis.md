## Tri Code Command List 文件總結


[未有直接 Source 錨點，待確認] 1. `check`: 檢查韌體檔案的簽章資訊。
[未有直接 Source 錨點，待確認] 2. `dump`: 從裝置中轉儲特定的韌體。
[未有直接 Source 錨點，待確認] 3. `erase`: 在裝置上清除特定的韌體。
[未有直接 Source 錨點，待確認] 4. `errorcode`: 顯示錯誤代碼的說明。
[未有直接 Source 錨點，待確認] 5. `help`: 列印幫助資訊。
[未有直接 Source 錨點，待確認] 6. `info`: 顯示裝置資訊。
7. `isp`: 進行 ISP 韌體更新。
8. `ispcs`: 對 ISP 韌體進行簽章。
9. `request`: 向控制管道發送 USB 設定封包。
10. `reset`: 重置裝置。
11. `status`: 顯示裝置狀態。
12. `system`: 執行系統操作。
