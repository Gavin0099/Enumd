
# Visual Studio 2017 Administrator 設定方式


1. 開始 --> 右鍵 Visual Studio 2017 捷徑 --> 開啟檔案位置
2. 右鍵 Visual Studio 2017 捷徑 --> 開啟檔案位置
3. 右鍵 `devend.exe` --> 疑難排解相容性
5. 選擇 `程式需要額外的權限` --> 下一步
6. 選擇 `測試程式開啟 visual studio 2017 看看是否有 administrator 權限` --> 下一步
7. 選擇 `是，為此程式儲存這些設定`


1. 透過右鍵 Visual Studio 2017 捷徑，可以找到其實際的檔案位置。這是設定 Administrator 權限的第一步。
2. 同樣地，右鍵 Visual Studio 2017 捷徑也可以直接開啟其檔案位置。
3. `devend.exe` 是 Visual Studio 2017 的一個相關元件。右鍵它並選擇 `疑難排解相容性` 可以進入設定 Administrator 權限的流程。
4. 選擇 `疑難排解程式` 進入設定流程。
5. 選擇 `程式需要額外的權限` 並點選 `下一步`，表示需要提升權限。
6. 選擇 `測試程式開啟 visual studio 2017 看看是否有 administrator 權限`，這是確認是否需要提升權限的關鍵步驟。
[未有直接 Source 錨點，待確認] 7. 如果測試通過，選擇 `是，為此程式儲存這些設定` 以儲存 Administrator 權限的設定。
