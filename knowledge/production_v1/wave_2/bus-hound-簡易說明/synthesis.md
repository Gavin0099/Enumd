# Bus Hound 簡易說明

Bus Hound 是一款可以抓取 USB 傳輸資料的工具。以下是它的主要功能:


- **CTL**: 代表 USB control transfer，是工具傳送給 USB 裝置的資料。
- **IN**: 代表 USB 裝置傳回來的資料。
- **Run & Stop**: 按下 Run 會持續抓取 USB 裝置的資料，按下 Stop 則會停止抓取。


選擇 Save 即可儲存抓取到的 USB 裝置資料。



  - **Capture Capacity**: 最多可以抓取的大小，如果超過就必須停止抓取並重新開始。
  - **Max Record Length**: 每個 packet 最多能抓取的資料大小。
- **Stop when**: 設定遇到什麼條件會停止抓取。
- **Phases to Capture**: 勾選要抓取的資料類型。
- **Columns to Display**: 勾選要顯示的資訊。


選擇要抓取的裝置，可以透過 Hardware ID 來識別。

以上就是 Bus Hound 的主要功能說明。如需更多資訊，請參考 [HP OCI APP](path/to/hp-oci-app.html) 和 [Clawdbot 自主代理建置與資安防禦全紀錄](path/to/-clawdbot-自主代理建置與資安防禦全紀錄.html) 中的相關內容。
