# GenesysLogic 整合 SDK 規範

客戶要求 GenesysLogic 的 ISP 工具整合第三方設備的 ISP 功能時，設備供應商提供的 SDK 應符合 ISP 工具所接受的介面與格式。

在 GenesysLogic 開發工具的過程中，設備供應商提供自行定義的介面與格式的 SDK，導致 GenesysLogic 整合時需要做額外的確認，更甚者需要多做處理才可以使用，拖延 GenesysLogic 開發的時程，進而影響產品開發進度。
