2021 年重點目標報告

## 目標概述
2021 年的重點目標是將 HP Enduser Tool、General Enduser Tool 和 Standard ISP Tool 整合成新的架構。在此過程中需要事先規劃好要測試的 HP Display Model 以及各種 code sign 方式，並在每週四的例會中檢視進度。

## 閱讀書籍
為了達成上述目標，計畫在 2 個半月內閱讀完以下兩本書籍:
1. [設計模式的解析與活用](https://www.books.com.tw/products/0010844576)
2. [大話設計模式](https://www.books.com.tw/products/0010291950)

## 技術學習
除了閱讀書籍外，也計畫學習並熟悉以下技術:
- [Design pattern](https://en.wikipedia.org/wiki/Design_pattern)
- C++11 常用部分

## 改善 LOG 系統
計畫嘗試加入 [SPD Log](https://github.com/SPDOfficial/SPDLog) (一個較完整的 Open source LOG 系統) 以取代原有的 LOG 系統。

## 流程圖繪製
使用 [draw.io](https://app.diagrams.net/) 繪製流程圖。

## 其他方法論
- 了解戴明環 (PDCA) 及 Six Sigma 等持續改善的方法論。
