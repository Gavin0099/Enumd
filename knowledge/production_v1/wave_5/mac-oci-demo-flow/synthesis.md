
# Mac OCI Demo flow


1. 在指定資料夾放好 Mac OCI Tool，並把 hub & scaler 前後版 bin 檔也放進去。[Mac OCI Tool](mac/mac-oci-demo-flow.html)
2. 開啟 Mac OCI Tool，會顯示以下資訊。[Mac OCI Tool](mac/mac-oci-demo-flow.html)
3. 開啟 Mac OCI Tool 時同時也用攝影機直播 demo Mac 筆電有跟 Z40 透過 Type-C 連接，並且可以顯示 Z40 螢幕為延伸螢幕，藉此移動 OCI Tool 來顯示 Z40 有 demo 筆電的訊號。[Mac OCI Tool](mac/mac-oci-demo-flow.html)
4. 更新 Hub & Scaler firmware。[Mac OCI Tool](mac/mac-oci-demo-flow.html)
5. 更新成功後修改 xml 或 ini 參數，把要 update 的 hub & scaler bin 檔檔名修改，重新開啟 OCI Tool，確認 Tool 有去讀取 xml 或 ini 參數來決定 update bin 檔。[Mac OCI Tool](mac/mac-oci-demo-flow.html)
6. 更改 XML 增加 Hub L2，再開啟 Tool 確認是透過 xml 來獲取資訊。[Mac OCI Tool](mac/mac-oci-demo-flow.html)
